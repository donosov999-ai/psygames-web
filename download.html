Как установить

Платформа-specific нюансы при первом запуске.

🍎 macOS (Apple Silicon)
  1. Распакуй PsyGames-macos-arm64.tar.gz (двойной клик в Finder)
  2. Перенеси PsyGames.app в /Applications
  3. Первый запуск: правый клик → Открыть → Открыть всё равно (обход Gatekeeper, приложение не подписано Apple Developer cert)
  4. Дальше открывается как обычное приложение из Launchpad / Dock

⚠ Если установлена старая версия — сначала Cmd+Q на текущей, потом замени в /Applications, иначе запустится cached.

🪟 Windows (10 / 11)
  1. Скачай PsyGames__x64-setup.exe (NSIS-installer) или .msi на выбор
  2. Запусти. Windows SmartScreen может предупредить — «Подробнее» → «Выполнить в любом случае» (не подписано Microsoft cert)
  3. Установится в %APPDATA%\Local\PsyGames\, ярлык в меню Пуск
  4. Требуется WebView2 Runtime — обычно установлен на Windows 10+ из коробки. Если нет — скачать здесь
🤖 Android (7.0+)
  1. Скачай PsyGames-android.apk на телефон
  2. Открой файл — Android спросит разрешение на установку из неизвестных источников. Разреши
  3. Тап «Установить»
  4. Запускай из app drawer как обычно

📝 APK подписан, повторные обновления сохраняют данные. Не доступен в Google Play — установка только через APK.

🌐 Веб-версия и iOS (PWA)
  1. Открой https://donosov999-ai.github.io/psygames-web/ в браузере (любом)
  2. На iPhone: Safari → «Поделиться» → «На экран Домой» — иконка PsyGames появится как обычное приложение
  3. На десктопе работает прямо во вкладке. Прогресс хранится в localStorage браузера.
