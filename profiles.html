Каталог всех 48 упражнений

Полная библиотека. Каждое упражнение — научно обоснованный нейропсихологический тренажёр. Фильтруй по категории.

🧠N-back: оперативная память
Удерживай поток стимулов и сравнивай с тем, что было N шагов назад. Золотой стандарт WM-тренинга.
Working memory
🧠Парные картинки
Открывай карточки и находи пары. Классика тренировки эпизодической памяти.
Visual memory
🧠Матрица памяти
Запомни паттерн подсвеченных клеток за 2 сек, потом воспроизведи. Зрительная WM.
Visual memory
🧠Запомни цифры (Digit Span)
Послушай/посмотри последовательность цифр и повтори (прямой/обратный). Stand the test of time.
Short-term memory
🧠Пары слов
Запомни ассоциированные пары (имя→фамилия). Тренировка вербальной памяти.
Verbal memory
🧠Мнемоника: порядок
Удерживай и воспроизводи последовательности из 6-12 элементов.
Sequence memory
🧠Reading Span
Читай предложения и одновременно запоминай слова. WM при чтении.
Working memory
🧠Корси: кубики
Повтори последовательность подсвеченных кубиков (прямую или обратную). Spatial WM.
Spatial memory
🧠OSpan: счёт+память
Решай арифметику и одновременно запоминай буквы. Operation Span — классика когнитивной нагрузки.
Working memory
🧠Spatial Span (обратный)
Повтори пространственную последовательность в обратном порядке.
Spatial WM
🧠Span: память на последовательности
Гибрид нескольких видов span-задач. Универсальная WM-тренировка.
Short-term memory
🧠Story Recall: понимание текста
Прочти короткую историю, потом вспомни как можно больше деталей (имена/числа/события).
Episodic memory
🎯Шульте: таблица
Найди числа 1-25 по порядку. Классика тренировки фокуса и расширения поля зрения.
Selective attention
🎯CPT: устойчивое внимание
Реагируй на все буквы кроме X в течение 4-12 минут. Sustained attention measure.
Sustained attention
🎯Найди отличия
Сравни две почти идентичные картинки и найди 5-10 отличий. Внимание к деталям.
Detail attention
🎯Visual Search: быстрый scan
Найди целевой объект среди дистракторов. Тренировка скорости сканирования.
Visual search
🎯Корректура: фокус
Найди опечатки или специфические буквы в тексте. Тренировка внимания к деталям.
Focus
🎯Reading the Mind in Eyes (RMET)
Определи эмоцию человека только по выражению глаз. Социальная когниция / эмпатия.
Social cognition
🎯Торможение (Inhibition)
Подавляй автоматические реакции на конфликтующие стимулы.
Inhibitory control
🧩Башня Лондона
Перемести шары в нужное положение за минимум ходов. Тренировка планирования.
Planning
🧩Ханойская башня
Перенеси башню из колец на другой штырь по правилам. Алгоритмическое мышление.
Problem solving
🧩Судоку 6×6
Заполни сетку цифрами 1-6 без повторов. Дедуктивная логика.
Logical reasoning
🧩Анаграммы
Составь слова из перемешанных букв. Вербальная гибкость.
Verbal flexibility
🧩Паттерны: мышление
Найди закономерность в последовательности и продолжи её. Inductive reasoning.
Pattern recognition
🧩SET: тройки признаков
Найди тройку карточек с уникальным сочетанием признаков. Multi-attribute reasoning.
Abstract reasoning
🧩Ментальная ротация
Поверни 3D-фигуру в уме и сравни с целевой. Spatial reasoning.
Spatial reasoning
🧩Соедини цепочку (Trail Making)
Соединяй точки чередуя цифры и буквы (1-А-2-Б...). Executive function.
Switching
🧩Переключение задач
Меняй правило ответа по сигналу. Cognitive flexibility.
Task switching
🧩WCST: правила
Угадай правило сортировки карточек, потом адаптируйся к смене правила. Wisconsin Card Sort.
Rule learning
🧩BART: риск-баллон
Накачивай баллон — больше насосов = больше денег, но он лопнет. Risk decision.
Risk-taking
🧩Iowa: 4 колоды
Выбирай карты из 4 колод. Две прибыльные long-term, две убыточные. Long-term strategy.
Long-term decision
🧩PRL: смена правил
Probabilistic Reversal Learning — учись правилу, потом перестраивайся когда оно меняется.
Reversal learning
🧩Phonemic Fluency
Назови как можно больше слов на определённую букву за 60 сек. Вербальная беглость.
Verbal fluency
⚡Мишени: реакция
Кликай по появляющимся мишеням как можно быстрее. Простая reaction time.
Reaction
⚡Выбор-реакция (Choice RT)
Выбирай правильную из 2-4 кнопок в ответ на стимул. Decision speed.
Choice speed
⚡SDMT: символ→цифра
Заменяй символы на цифры по ключу. Золотой стандарт processing speed (особенно для возраста).
Processing speed
⚡Математический спринт
Решай арифметику на скорость в течение 60-120 сек.
Mental math
⚡Считалка: счёт
Простой устный счёт + проверка ответов. Бытовая арифметика.
Math
⚡Числовые пары
Найди числа которые в сумме дают целевое (10, 20...).
Math
⚡Конфликт внимания
Гибрид Stroop + Flanker. Тренировка торможения интерферирующих стимулов.
Inhibition
⚡Stroop: торможение
Назови цвет шрифта, игнорируя значение слова. Классический test of inhibition.
Inhibition
⚡Эмоциональный Stroop
Stroop с эмоционально-окрашенными словами. Регуляция внимания при стрессе.
Emotional inhibition
⚡Go / No-Go
Нажимай на «go»-стимулы и тормози на «no-go». Прямое измерение тормозного контроля.
Inhibition
⚡Stop-Signal
Реагируй, но останавливай ответ если прозвучит стоп-сигнал. Reactive inhibition.
Inhibition
⚡Фланкер: стрелки
Отвечай на центральную стрелку, игнорируя боковые. Selective attention под помехами.
Inhibition
⚡Торможение базовое
Простой тренажёр inhibitory control для начинающих.
Inhibition
⚡Conflict tasks (mixed)
Смешанная батарея конфликт-задач (Simon, Flanker, Stroop) — gold standard executive measure.
Mixed inhibition
⚡Simon: цвет vs позиция (48-я игра!)
Цветной квадрат появляется слева/справа от центра. Жми ЛЕВУЮ если СИНИЙ, ПРАВУЮ если КРАСНЫЙ — по ЦВЕТУ, не по позиции. Когда позиция не совпадает с правильной стороной ответа — мозг тормозит. Simon Effect = разница RT.
Inhibition · spatial conflict
